# Kanban Board Application

A modern Kanban board built with Next.js, React, and TypeScript.

## Features
- Drag and drop kanban cards
- Task management
- Responsive design
- Component library with Storybook

## Getting Started

### Frontend
```bash
cd frontend
pnpm install
pnpm dev
```

### Storybook
```bash
cd frontend
pnpm storybook
```
